02 · Curriculum

What You Will Learn

01
Data Structures Foundation
Working with AnnData, managing spatial metadata and layers, structuring complex multi-sample datasets for downstream analysis.
02
Preprocessing — Robust by Design Core
Outlier detection via Median Absolute Deviation (MAD), automated QC pipelines, tissue-aware filtering, off-tissue spot removal, and gene renaming with safe ID fallback.
03
Clustering Strategy Core
Resolution selection via the silhouette method, structural heuristics for stable clustering, and principled interpretation of cluster results.
04
Data Integration Advanced
Multi-sample integration, batch correction strategies, and cross-slide harmonization for cohort-level analysis.
05
Spatial Analysis Advanced
Spatial visualization across multiple slides, QC plots, interactive region selection, and colocalization analysis using a custom method.

05 · Environment

Running with Docker 🐳

Start a JupyterLab instance with the full environment preconfigured — no dependency headaches.

1
Pull the image
shell
docker pull pedrovp161/spatialcourse:latest
2
Run the container
shell
docker run -it \
  -p 8888:8888 \
  pedrovp161/spatialcourse:latest \
  jupyter lab \
    --ip=0.0.0.0 \
    --no-browser \
    --allow-root \
    --NotebookApp.token=''
3
Open in your browser
url
http://localhost:8888
